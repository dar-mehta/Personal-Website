Repleno Charger (Increases Battery's Life Span)

I co-founded Repleno Labs in order to develop an innovative mobile charger that will help increase battery life span. My peers and I decided to pursue this challenge because modern day cellphone batteries deteriorate noticeably in just a matter of months thus holding less charge per cycle. An android app automatically calculates how to safely charge the battery using an algorithm and sends this data to the charger via bluetooth. The user is also able to set alarms on the Repleno Charger android app to indicate when he/she requires the phone to be fully-charged (the user can also turn off the charger's functionality through the app in order to charge it normally if he/she is in a rush). As the Chief Technology Officer, I am responsible for the circuit design, component selection, and pcb design, testing & troubleshooting.

    Dar Mehta, Yury Zhuk, Vik Goel, Jay Koshy

    Circuit Design, Component Selection, PCB Design (Upverter)

Gesture-Controlled Robotic Arm
  View Code

Collaborated with three fellow mechatronics engineering students to create a gesture-controlled robotic arm controlled by the Myo Armband. The purpose was to create a robotic arm with an intuitive, easy to use control system. The device was used to pick up a beaker, move the beaker towards a glass, and pour the beaker's contents into the glass. The Monobrick Bluetooth library for C++ was modified for the purpose of this project to send direct motor and sensor commands to the Lego NXT brick.

Wheg Stair Climbing Robot

This robot utilizes a unique form of locomotion: actuated whegs. Whegs (wheel-legs) provide the robot with the ability to clear obstacles but in turn, cause a bumpy ride on smooth terrain. By utilizing a complex actuated wheg mechanism, this robot is able to transform between wheel-mode (used for driving on smooth terrain) and leg-mode (used for clearing obstacles such as climbing stairs) thus combining the benefits of traditional wheels and the wheg mechanism. The robot utilizes a bluetooth connection between an android phone and its arduino microcontroller. Designed in solidworks and fabricated using 3D printed parts and laser-cut aluminum.

    Dar Mehta, Richard Li, Nikhil Keswani

    Solidworks, Arduino, Bluetooth

Cycle - BattleHack Toronto 2015
  View Code

Designed & built an NFC-enabled bike lock using an Arduino Uno and a PN532 NFC Shield. Designed and built an android application that enables bike owners to rent out their bikes, allowing the owner to set the hourly rate and put up a descripton and picture of the bike. The same application can be used by customers to see all the bikes that have been put up for rent in their area and click on each one for more info. Once a user selects a bike to rent, the user's PayPal account will be authenticated so that they can be charged at the hourly rate through their PayPal account. Once the user is authenticated, that particular phone will be given permission to unlock that bike's lock using NFC.

Receipt Digitizer - Hack the North Waterloo 2015
  View Code

Built an Android application to automate budgeting for consumers. Users are able to take a picture of their receipts. The Android app, using Google's Tesseract OCR library, converts the image into useful text such as the store's name, and isolates product name and price. This data is then exported to an excel spreadsheet for easy budgeting and hassle-free tracking of receipts.
